# 📚 Aula Curso LLM

## O que é **MCP**? 🤔

**MCP** (**Modelos de Conversação Pretreinados**) são grandes modelos de linguagem que já foram treinados em vastas quantidades de texto antes de serem adaptados para tarefas específicas. Eles são capazes de:

- 📖 **Entender** contextos complexos;
- ✍️ **Gerar** respostas coerentes e criativas;
- 🔧 **Ser ajustados** (fine‑tuned) para domínios como atendimento ao cliente, educação, programação, entre outros.

### Por que usar MCP? 🚀

1. **Economia de tempo** – Não é necessário treinar do zero;
2. **Performance** – Resultados de alta qualidade em múltiplas línguas;
3. **Flexibilidade** – Pode ser integrado a aplicativos, bots, assistentes virtuais, etc.

### Como funciona? 🛠️

1. **Pré‑treinamento** – O modelo aprende padrões gerais de linguagem a partir de bilhões de palavras;
2. **Ajuste fino** – Você alimenta o modelo com dados específicos da sua aplicação;
3. **Implantação** – O modelo está pronto para ser usado via API ou localmente.

### Exemplos de uso 🌟

- 🤖 **Chatbots** que respondem perguntas de suporte;
- 📚 **Tutoriais interativos** que explicam conceitos em tempo real;
- 🧑‍💻 **Assistentes de codificação** que sugerem trechos de código.
